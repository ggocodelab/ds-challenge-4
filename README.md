# Sobre
Repositório criado para a entrega do desafio de conclusão do módulo 4 do curso [Java Spring Professional](https://devsuperior.com.br/colecao-java-spring). Trata-se de 4 consultas elaboradas com critérios pré-definidos pelo intrutor.

# Recursos
* Java 25
* Maven 4.0.0
* Spring Boot 4.0.1
* Spring Tool Suite 4 (IDE)
* H2

# Requisitos
1. Report de vendas  
[✔] Listagem paginada contendo id, data, quantia vendida e nome do vendedor.  
[✔] Funcionalidade de filtragem por range de datas sendo opcionais tanto a data inicial quanto a data final.  
[✔] Filtragem por trecho do nome do vendedor.  
[✔] Se a data final não for informada, considerar a data atual do sistema.  
[✔] Se a data inicial não for informada, considerar a data de 1 ano antes da data final.  
[✔] Se o nome não for informado, considerar campo vazio.  

2. Report de vendas por vendedor  
[✔] Listagem contendo: nome do vendedor e soma de vendas do vendedor no período informado.  
[✔] Todos os requisitos da consulta anterior, exceto o primeiro.
